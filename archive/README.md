# Archive

Historical artifacts from earlier phases of OllamaBob. Preserved for reference
but no longer part of the active project.

## Contents

| Path | What it is | Phase |
|------|-----------|-------|
| `phase0/` | Pre-implementation investigations — num_ctx benchmarks, tool-call wire format samples, jq bundle viability check | Pre-V1 (April 2026) |
| `future_features/` | Parked feature ideas (generate_images prompt list) | Ongoing — reactivate when scheduled |
| `OLAMMABOB_PROMPT.md` | Original V1 kickoff prompt handed to the first implementation agent | V1 |
| `OLLAMA_CLAUDE_V2.5_plan.md` | Autonomous orchestration plan used to ship V2.5 "Make Bob Sing" | V2.5 |

## Why these are archived, not deleted

Each of these shaped a real decision the project is still living with. Keeping
them in the repo means the reasoning behind the decision stays recoverable
(via git history and file content) even after the originating doc is no longer
relevant day-to-day.
